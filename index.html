Basics

Every html page must contain 5 tags:

  • DOCTYPE defines the document type;
  • html says where the document starts and ends;
  • head contains metadata/information about the document;
  • title located in header, defines name of the page;
  • body defines the document's body or content.

Most of elements have an opening tag and a closing tag.
Some elements have only opening tag, for instance br, link.

Lists

Lists can be either:

  • unordered listis used to group a set of related items in no particular order,
  • ordered list is used to group a set of related items in a specific order,
  • description list is used to display name/value pairs such as terms and definitions.

Unordered (bulleted) lists are used when a set of items can be placed in any order, fx, a shopping list. Bullet styles can be adjusted in CSS.

  • bread,
  • milk,
  • chocolate.

Ordered (numbered) lists are used to display a list of items that should be in a specific order, fx, steps to get your file on Github:

  1. Add or change file in Explorer
  2. Save
  3. Open Source Control
  4. Stage Changes by pressing +
  5. Commit by pressing V and describe fx "Adding index"
  6. In Terminal "git push origin main"

Description lists associate specific names and values within a list, fx, an ingredient list. Can be made also in CSS with none.

To bake a bread, you need:
flour
milk
butter
To bake a cake, add:
eggs
sugar

Nested list (multilevel list) is a list item that contains another entire list, fx, table of contents.

Links

Hyperlinks or links can be absolute or relative.

Absolute links contain a full web address, fx to google.com.

Relative links are local links that can be scheme-relative (//example.com), origin-relative (en/uploads/images) or directory relative (./p).

Use title to tell the user something about the link.

Links can be opened in a tab, window or iframe. Use target to define where to open the link:

  • _self in current setting (default),
  • _blank a new tab unless the browser has a custom configuration,
  • _parent,
  • _top.

Links can be used for contact details such as email and phone number.
Send me an e-mail or call me anytime!

In CSS use

  • a:link - never visited,
  • a:visited - visited,
  • a:active - on click,
  • a:hover - on rolover.
